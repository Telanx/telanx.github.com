进程和线程

1.进程和线程特点
进程:子进程复制主进程内存空间，子进程崩溃不会影响其它进程，更稳定,相应代价大。
线程:子线程共享同一片内存区域，因此会有竞争;一个线程崩溃会影响整个进程。
